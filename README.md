# VueJS Test Project

Scaffolded minimal Vite + Vue 3 project for the frontend test.

Quick start (Windows):

```powershell
cd "[location your project]"
npm install
npm run dev
```

Open http://localhost:5173

What is included:
- Vue 3 + Vue Router
- Simple in-memory products store (create/edit/delete)
- `src/views/Products.vue`, `src/components/*`
